This lightweight Python SDK lets you call [PyroMind AI](https://pyromind.ai/) platform APIs to manage training workflows, Jupyter instances, inference jobs, EchoMind, and more.

Use the following guides by task:

## Get started

- [Installation](/docs/sdk/installation) - Install the SDK from PyPI and configure your API key and client

## Platform operations

- [Create a development environment](/docs/sdk/jupyterlab) - Create a Jupyter instance and get its access URL
- [Run training workflows](/docs/sdk/training-workflow) - Create, run, and wait for training jobs to complete
- [Create inference instances](/docs/sdk/inference) - Create and manage inference instances
- [Upload, view, and download files](/docs/sdk/storage) - Common object storage operations
- [Create and validate custom nodes](/docs/sdk/custom-nodes) - Convert Python functions to nodes, validate YAML, and register nodes
- [EchoMind automated training](/docs/sdk/echo-mind) - Create and manage EchoMind crawler instances
- [View user information](/docs/sdk/user-profile) - Account details, storage usage, and SSH keys
