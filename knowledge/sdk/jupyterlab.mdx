## Prerequisites

- Complete [SDK installation and configuration](/docs/sdk/installation)
- Have a valid PyroMind API key

## Create an instance

```python
from pyromind_sdk import PyroMindAPIClient
from pyromind_sdk.client.models import JupyterRequest, ResourceConfig

client = PyroMindAPIClient(api_key="your-api-key")

jupyter = client.jupyter.create(
    JupyterRequest(
        name="my-notebook",
        resources=ResourceConfig(cpu="4", memory="16Gi", gpu="1", gpu_card="L40S")
    )
)
print(f"Jupyter ID: {jupyter.id}, URL: {jupyter.url}")
```

## `JupyterRequest` parameters

| Parameter | Required | Type | Description |
|------|------|------|------|
| `name` | Yes | `str` | Display name of the instance |
| `resources` | Yes | `ResourceConfig` | CPU / memory / GPU configuration |

## API quick reference

| Method | Input | Output | Description |
|------|------|------|------|
| `list()` | - | `List[JupyterResponse]` | List all Jupyter instances |
| `create(request)` | `JupyterRequest` | `JupyterResponse` | Create a new instance |
| `get_instance(jupyter_id)` | `str` | `JupyterResponse` | Get instance details |
| `update(jupyter_id, request)` | `str`, `JupyterRequest` | `JupyterResponse` | Update instance configuration |
| `delete(jupyter_id)` | `str` | `None` | Delete an instance |
| `pause(jupyter_id)` / `resume(jupyter_id)` | `str` | `JupyterResponse` | Pause / resume |

## Response fields

| Field | Type | Description |
|------|------|------|
| `id` | `str` | Instance ID |
| `name` | `str` | Instance name |
| `status` | `str` | Current status |
| `password` | `Optional[str]` | Access password |
| `resources` | `Optional[ResourceConfig]` | Allocated resources |
| `url` | `Optional[str]` | Jupyter access URL |
| `created_at` | `Optional[datetime]` | Created time |
| `updated_at` | `Optional[datetime]` | Updated time |
