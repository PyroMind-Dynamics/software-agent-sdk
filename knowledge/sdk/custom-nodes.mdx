## Wrap a Python function as a node

```python
from pyromind_sdk.nodes.function_call_wrapper import create_node_from_function

@create_node_from_function(
    name="my_custom_node",
    description="Processes input data",
    category="data-processing"
)
def process_data(input_text: str, threshold: float = 0.5) -> dict:
    return {"result": "processed", "value": len(input_text)}
```

## Execute a Python node at runtime

```python
from pyromind_sdk.nodes.python_function_executor import execute_python_node

result = execute_python_node(
    source_code="print('hello')",
    node_type="python"
)
```

## Convert a Python function to YAML configuration

```python
from pyromind_sdk.nodes.python_to_yaml import python_function_to_yaml_config

def my_func(input: str) -> str:
    return input.upper()

yaml_config = python_function_to_yaml_config(my_func)
# yaml_config 可保存为 .yaml 文件，并通过 studio.create_node() 注册
```

## Validate and load YAML node definitions

```python
from pyromind_sdk.nodes.yaml_loader import load_yaml_node
from pyromind_sdk.nodes.node_validator import validate_node_config

node_config = load_yaml_node("path/to/node.yaml")
validate_node_config(node_config)
```

## Register in Studio

Use `client.studio` to register nodes to the platform:

| Method | Description |
|------|------|
| `create_node(...)` | Register a custom node (`yaml_path` / `yaml_content` + options) |
| `reload_nodes(node_name=None)` | Reload YAML node definitions |
| `delete_node_by_name(node_name)` | Delete a custom node by name |
| `move_node(node_name, source_file_path)` | Move a node source file |
| `get_node_info(names=None)` | Get node definition details |

## CLI tool

```bash
python -m pyromind_sdk.python_function_to_yaml_cli
```

Convert Python functions into YAML node definitions. For more CLI commands, see [More SDK information](/docs/sdk/more#cli-tools).
