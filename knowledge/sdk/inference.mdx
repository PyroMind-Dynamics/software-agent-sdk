## Prerequisites

- Complete [SDK installation and configuration](/docs/sdk/installation)
- Upload your model to Storage or ensure it is accessible via `model_path`

## Query available frameworks and images

```python
from pyromind_sdk import PyroMindAPIClient

client = PyroMindAPIClient(api_key="your-api-key")

frameworks = client.inference.get_framework()
images = client.inference.get_inf_image(frameworks[0])
print(f"Frameworks: {frameworks}")
print(f"Images: {images}")
```

## Create an inference instance

```python
from pyromind_sdk.client.models import InferenceJobRequest, ResourceConfig

job_id = client.inference.create(
    InferenceJobRequest(
        model_path="/workspace/models/Qwen/Qwen3-0.6B/",
        model_name="glm-5",
        inference_framework=frameworks[0],
        inf_image=images[0],
        resources=ResourceConfig(cpu="4", memory="32Gi", gpu=1, gpu_card="L40S"),
        name="my-inference",
    )
)
print(f"Job ID: {job_id}")
```

## Check instance status

```python
job = client.inference.get_job(job_id)
print(f"Status: {job.status}")
print(f"Endpoint: {job.endpoint_url}")
```

## List, update, and delete

```python
jobs = client.inference.list()
for j in jobs:
    print(f"{j.id} - {j.name} - {j.status}")

client.inference.pause(job_id)
client.inference.resume(job_id)
client.inference.delete(job_id)
```

## `InferenceJobRequest` parameters

| Parameter | Required | Type | Description |
|------|------|------|------|
| `model_path` | Yes | `str` | Model path |
| `inference_framework` | No | `str` | Framework name (from `get_framework()`) |
| `resources` | Yes | `ResourceConfig` | CPU / memory / GPU configuration |
| `name` | Yes | `str` | Display name of the job |
| `inf_image` | Yes | `str` | Inference image (from `get_inf_image()`) |
| `model_name` | Yes | `str` | Override model name |
| `model_length` | No | `int` | Model context length |

## API quick reference

| Method | Description |
|------|------|
| `list()` | List all inference instances |
| `create(request)` | Create an inference instance and return `job_id` |
| `get_job(job_id)` | Get job details |
| `update(job_id, request)` | Update job configuration |
| `delete(job_id)` | Delete a job |
| `pause(job_id)` / `resume(job_id)` | Pause / resume a job |
| `get_framework()` | List available inference frameworks |
| `get_inf_image(framework)` | List available inference images |

## Response fields (`InferenceJobResponse`)

| Field | Type | Description |
|------|------|------|
| `id` | `str` | Job ID (`job_id` in the raw response, exposed as `id` through an alias) |
| `name` | `str` | Job name |
| `model_path` | `str` | Model path |
| `inference_framework` | `Optional[str]` | Inference framework |
| `image` | `Optional[str]` | Image |
| `status` | `str` | Current status |
| `uid` | `Optional[str]` | User ID |
| `resources` | `Optional[ResourceConfig]` | Allocated resources |
| `endpoint_url` | `Optional[str]` | Inference endpoint |
| `created_at` | `Optional[datetime]` | Created time |
| `updated_at` | `Optional[datetime]` | Updated time |
| `model_name` | `Optional[str]` | Model name |
| `model_length` | `Optional[int]` | Model context length |
| `inf_image` | `Optional[str]` | Inference image |
