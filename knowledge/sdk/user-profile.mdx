## Prerequisites

- Complete [SDK installation and configuration](/docs/sdk/installation)

## View user information

```python
from pyromind_sdk import PyroMindAPIClient

client = PyroMindAPIClient(api_key="your-api-key")

resp = client.profile.get_user_info()
print(f"Logged in: {resp.is_logged_in}")
print(f"Username: {resp.user.username}")
print(f"Email: {resp.user.email}")
```

To query your credit balance:

```python
resp = client.profile.get_user_info(credit_info=True)
print(f"Credit: {resp.user.available_credit} {resp.user.currency}")
```

## Get access key

```python
access_key = client.profile.get_access_key()
print(f"Access Key: {access_key}")
```

## View storage credentials

```python
storage_info = client.profile.get_storage_info()
print(f"Endpoint: {storage_info.url}")
print(f"Access Key: {storage_info.access_key}")
```

<Note>
To view storage usage (used / total), use the module-level function `pyromind_sdk.client.storage.get_storage_info()`. It returns a `dict` containing fields such as `used_size`, `human_used_size`, `total_size`, and `human_total_size`.
</Note>

```python
from pyromind_sdk.client.storage import get_storage_info

info = get_storage_info()
print(f"Used: {info['human_used_size']} / Total: {info['human_total_size']}")
```

## Manage SSH public keys

```python
from pyromind_sdk.client.models import UserPubKeyRequest

client.profile.add_key(UserPubKeyRequest(name="my-laptop", key="ssh-ed25519 AAAA..."))
keys = client.profile.list_keys()
for k in keys:
    print(f"{k.id} - {k.name}: {k.pub_key}")
```

## API quick reference

| Method | Input | Output | Description |
|------|------|------|------|
| `get_user_info(credit_info=False)` | `bool` | `ProfileUserInfoResponse` | Get user information (returns credit fields when `credit_info=True`) |
| `get_access_key()` | - | `str` | Get access key |
| `get_storage_info()` | - | `ProfileStorageInfoResponse` | Get storage credentials |
| `add_key(request)` | `UserPubKeyRequest` | `bool` | Add an SSH public key |
| `list_keys()` | - | `List[UserPubKey]` | List SSH public keys |

## Response fields

### `ProfileUserInfoResponse`

| Field | Type | Description |
|------|------|------|
| `is_logged_in` | `bool` | Whether the user is logged in (JSON field name: `isLoggedIn`) |
| `user` | `ProfileUserInfo` | User details (nested object; access fields with `resp.user.xxx`) |

### `ProfileUserInfo`

| Field | Type | Description |
|------|------|------|
| `username` | `Optional[str]` | Username |
| `email` | `Optional[str]` | Email |
| `full_phone_num` | `Optional[str]` | Phone number |
| `uid` | `Optional[str \| int]` | User ID |
| `group_id` | `Optional[int]` | User group ID |
| `user_tier` | `Optional[str]` | User tier |
| `avatar_url` | `Optional[str]` | Avatar URL |
| `avatar_info` | `Optional[AvatarInfo]` | Avatar details |
| `credit_amount` | `Optional[int \| float \| str]` | Total credit amount |
| `available_credit` | `Optional[str]` | Available credit |
| `cash_balance` | `Optional[str]` | Cash balance |
| `currency` | `Optional[str]` | Currency |
| `account_restricted` | `Optional[bool]` | Whether the account is restricted |

### `ProfileStorageInfoResponse`

| Field | Type | Description |
|------|------|------|
| `access_key` | `str` | Storage access key |
| `secret_key` | `Optional[str]` | Storage secret key |
| `url` | `Optional[str]` | Storage endpoint URL |
| `uid` | `Optional[str]` | User ID |
