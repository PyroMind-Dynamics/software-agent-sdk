## Prerequisites

- Complete [SDK installation and configuration](/docs/sdk/installation)

## Typical workflow

1. **Create an instance** - Call `client.echomind.create()` to start a crawling task
2. **Check status** - Use `get_job()` or `list()` to monitor runtime status
3. **Pause / resume** - Call `pause()` / `resume()` as needed
4. **Clean up resources** - Call `delete()` when finished

## Create an EchoMind instance

```python
from pyromind_sdk import PyroMindAPIClient
from pyromind_sdk.client.models import EchoMindJobRequest, ResourceConfig

client = PyroMindAPIClient(api_key="your-api-key")

job_id = client.echomind.create(
    EchoMindJobRequest(
        name="my-echomind",
        api_url="https://generativelanguage.googleapis.com",
        api_mode="gemini",
        origin_model="gemini-1.5-flash",
        access_key="your-access-key",
        training_model="my-training-model",
        training_batch_size=32,
        trajectory_buffer_size=1000,
        time_per_round=60,
        training_round=100,
        training_save_path="/data/training/echomind",
        resources=ResourceConfig(cpu="4", memory="16Gi"),
        execution_mode="manual",
        custom_runtime_script_path="/workspace/trigger/say_hello.py",
    )
)
print(f"EchoMind Job ID: {job_id}")
```

## List instances and check status

```python
instances = client.echomind.list()
for job in instances:
    print(f"{job.job_id} - {job.name} - {job.status}")

job = client.echomind.get_job(job_id)
print(f"Status: {job.status}")
```

## Pause, resume, and delete

```python
client.echomind.pause(job_id)
client.echomind.resume(job_id)
client.echomind.delete(job_id)
```

## `EchoMindJobRequest` parameters

| Parameter | Required | Type | Description |
|------|------|------|------|
| `name` | Yes | `str` | Display name of the instance |
| `api_url` | Yes | `str` | Upstream model API URL |
| `api_mode` | Yes | `str` | API mode, one of `openai` / `gemini` / `anthropic` (default `openai`) |
| `origin_model` | Yes | `str` | Source model name (for example, `gemini-1.5-flash`) |
| `access_key` | Yes | `str` | Access key used to call the upstream API |
| `training_model` | Yes | `str` | Training model name |
| `training_batch_size` | Yes | `int` | Training batch size |
| `trajectory_buffer_size` | Yes | `int` | Trajectory buffer size |
| `time_per_round` | Yes | `float` | Duration per round in seconds |
| `training_round` | Yes | `int` | Number of training rounds |
| `training_save_path` | Yes | `str` | Path to save training outputs |
| `resources` | Yes | `ResourceConfig` | CPU / memory / GPU configuration |
| `execution_mode` | Yes | `str` | Execution mode, `manual` / `automatic` (default `manual`) |
| `custom_runtime_script_path` | Yes | `str` | Path to the custom runtime script |

## API quick reference

| Method | Input | Output | Description |
|------|------|------|------|
| `list()` | - | `List[EchoMindJobResponse]` | List all EchoMind instances |
| `create(request)` | `EchoMindJobRequest` | `str` (job_id) | Create an EchoMind instance |
| `get_job(job_id)` | `str` | `EchoMindJobResponse` | Get instance details |
| `update(job_id, request)` | `str`, `EchoMindJobRequest` | `EchoMindJobResponse` | Update instance configuration |
| `delete(job_id)` | `str` | `None` | Delete an instance |
| `pause(job_id)` / `resume(job_id)` | `str` | `EchoMindJobResponse` | Pause / resume |
