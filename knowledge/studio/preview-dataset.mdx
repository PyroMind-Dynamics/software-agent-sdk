## 使用 Dataset Validator 预览

**Dataset Processing** 工作流内置 **Dataset Validator** 节点，可在训练前校验数据并生成可视化预览：

1. 确认 **Dataset Kind Config Builder** 与 **Dataset Config Builder** 已正确配置字段映射与数据路径。
2. 将 `dataset_config` 输出连接到 **Dataset Validator** 节点。
3. 运行工作流，查看节点输出：
   - `metrics_yaml` — 数据集统计指标（样本数、字段完整性等）
   - `output_preview_path` — HTML 预览文件路径（默认 `/workspace/datasets/preview.html`）

## Validator 配置项

| 参数 | 说明 |
|------|------|
| `limit` | 预览样本数量上限，`0` 表示不限制 |
| `check_reasoning` | 是否校验推理链格式（如 `<<...>>` 标记） |
| `verbose` | 是否输出详细校验日志 |
| `preview_html_path` | 预览 HTML 文件的保存路径 |

## 检查清单

| 检查项 | 期望结果 |
|--------|----------|
| 样本数量 | 与预期一致，无异常过少 |
| 标签格式 | 与下游 SFT / DPO / GRPO 节点要求匹配 |
| 异常值 | 无空标签、超长空白或损坏文件 |
| 划分方式 | 训练 / 验证集划分符合实验设计 |

## 常见问题

- **路径错误**：确认 Storage 挂载正常，路径与 [分布式存储](/zh/docs/basic/storage) 中一致。
- **字段不匹配**：回到 **Dataset Kind Config Builder** 调整映射，或修正源数据格式。
- **预览为空**：检查过滤规则是否过于严格，导致样本被全部剔除。
