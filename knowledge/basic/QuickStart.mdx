本节涵盖 PyroMind **控制台（Web UI）** 的日常操作，帮助您完成账户配置、创建实例并管理各产品。若需通过 Python SDK 以编程方式调用 API，请参阅 [PyroMind SDK](/zh/docs/sdk/index)。

## 推荐阅读顺序

1. [注册和登录](/zh/docs/basic/signup_login) — 注册账户并登录 PyroMind 控制台
2. [用户配置](/zh/docs/basic/user_profile) — SSH 公钥、API Key 与资源配额
3. [JupyterLab](/zh/docs/jupyterlab/how-to-use) — 创建实例并通过 Web 或 SSH 连接
   - [SSH 组合用户名登录](/zh/docs/jupyterlab/ssh-combined-login) — 共享实例多用户 SSH 配置
4. [Studio](/zh/docs/studio/how-to-use) — 可视化训练工作流
5. [Inference](/zh/docs/basic/inference) — 模型推理任务部署与管理
6. [EchoMind](/zh/docs/basic/echo-mind) — 全自动数据抓取
7. [分布式存储](/zh/docs/basic/storage) — 上传、浏览、下载文件，挂载到实例
