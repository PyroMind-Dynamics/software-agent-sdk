This section covers day-to-day operations in the PyroMind **Console (Web UI)** and helps you complete account setup, create instances, and manage each product. If you need to call APIs programmatically through Python, see the [PyroMind SDK](/docs/sdk/index).

## Recommended reading order

1. [Sign up and log in](/docs/basic/signup_login) — Register your account and sign in to the PyroMind Console
2. [User profile](/docs/basic/user_profile) — SSH public keys, API key, and resource quotas
3. [JupyterLab](/docs/jupyterlab/how-to-use) — Create instances and connect via Web or SSH
   - [SSH combined username login](/docs/jupyterlab/ssh-combined-login) — Multi-user SSH setup for shared instances
4. [Studio](/docs/studio/how-to-use) — Visual training workflows
5. [Inference](/docs/basic/inference) — Deploy and manage model inference jobs
6. [EchoMind](/docs/basic/echo-mind) — Fully automated data crawling
7. [Distributed storage](/docs/basic/storage) — Upload, browse, download files, and mount them to instances
